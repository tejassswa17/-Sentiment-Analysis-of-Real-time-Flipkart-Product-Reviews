#  Sentiment Analysis using Machine Learning

##  Overview

This project performs **sentiment analysis on text data** using Machine Learning.
It classifies input text into different sentiments such as **positive, negative, or neutral**.

---

##  Project Files

* `sentiment_task.ipynb` → Data preprocessing, training, and evaluation
* `sentiment_model.pkl` → Trained ML model
* `tfidf_vectorizer.pkl` → Text vectorization using TF-IDF
* `data.csv` → Dataset used for training

---

##  Technologies Used

* Python
* Scikit-learn
* Pandas, NumPy
* TF-IDF Vectorization

---

##  How It Works

1. Text data is cleaned and preprocessed
2. TF-IDF vectorizer converts text into numerical features
3. Machine Learning model is trained on the data
4. Model predicts sentiment based on input text

---

##  Model Pipeline

Text → Preprocessing → TF-IDF → Model → Sentiment Prediction

---

##  Features

✔ Text preprocessing and cleaning
✔ TF-IDF based feature extraction
✔ Trained classification model
✔ Easy to reuse saved model (`.pkl` files)

---

##  Use Case

* Social media sentiment analysis
* Review classification
* Feedback analysis

---

## Conclusion

This project demonstrates how Machine Learning can be used to analyze text data and extract meaningful insights through sentiment classification.
